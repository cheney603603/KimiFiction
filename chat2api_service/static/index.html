📖 使用说明

  1. 从上方选择要使用的 AI 助手
  2. 点击"打开登录页面"按钮，浏览器会自动打开对应 AI 网站
  3. 在打开的浏览器窗口中完成登录
  4. 登录成功后，点击"确认已登录"按钮
  5. 然后在下方输入框中输入消息，点击发送即可

🔌 API 接口

你也可以通过代码直接调用 API：

# Python 示例 import requests # 发送消息（指定 AI 提供商） response = requests.post( "http://localhost:8000/api/kimi/chat", # 或 deepseek, doubao, yuanbao json={"message": "你好！", "timeout": 120} ) print(response.json()) # 检查登录状态 status = requests.get("http://localhost:8000/api/status") print(status.json()) # 获取支持的 AI 提供商列表 providers = requests.get("http://localhost:8000/api/providers") print(providers.json())
