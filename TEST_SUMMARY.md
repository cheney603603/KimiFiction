## 可能的问题和解决方案

### 问题1：角色设计阶段执行失败

**症状**：
- 工作流页面角色设计阶段显示错误
- 后端日志显示角色设计失败

**可能原因**：
1. LLM配置未设置或配置错误
2. 请求超时
3. API调用失败
4. 数据解析失败

**解决方案**：
1. 检查LLM配置是否正确
2. 增加timeout参数
3. 查看后端日志了解详细错误
4. 重新执行角色设计阶段

### 问题2：角色同步到数据库失败

**症状**：
- 角色设计成功，但角色管理页面看不到角色
- 后端日志显示同步失败

**可能原因**：
1. 数据库连接问题
2. 角色数据格式不正确
3. 保存API调用失败

**解决方案**：
1. 检查数据库连接
2. 查看后端日志找到具体错误
3. 在角色管理页面手动添加角色
4. 重新执行角色设计阶段

### 问题3：章节生成失败

**症状**：
- 章节写作页面点击"生成章节"后没有反应
- 或显示"生成失败"

**可能原因**：
1. LLM配置未设置
2. timeout太短
3. 请求超时
4. 章节大纲不存在

**解决方案**：
1. 检查LLM配置
2. 增加timeout参数（建议15-30分钟）
3. 确保已完成大纲设计阶段
4. 查看后端日志了解详细错误

### 问题4：章节保存失败

**症状**：
- 点击"保存到数据库"失败
- 提示"保存失败，请重试"

**可能原因**：
1. API调用失败
2. 数据库错误
3. 内容格式不正确

**解决方案**：
1. 检查浏览器控制台的错误信息
2. 查看后端日志
3. 检查API端点是否正常
4. 重试保存

### 问题5：阅读页面显示空白

**症状**：
- 访问阅读页面显示空白或"加载中..."

**可能原因**：
1. 章节未保存到数据库
2. API请求失败
3. 数据格式不正确

**解决方案**：
1. 确认章节已保存到数据库
2. 检查API端点
3. 查看后端日志
4. 刷新页面重试
