# Phase 4 参考：正文写作风格指南

## 写作核心理念

网络小说是**产品**，不是纯文学创作。核心原则：

1. **读者时间宝贵** — 每一段都要有信息量或情绪价值
2. **节奏是生命** — 不能水文，不能拖沓
3. **代入感优先** — 让读者"成为"主角
4. **爽点要写透** — 重要爽点不能一笔带过

## 网络小说正文写作黄金法则

### 法则1：开篇即冲突

**绝对不能在开头大段铺世界观**，读者前三段没看到冲突/悬念，直接弃书。

正确示例：
> "我穿越了。" 林逸看着自己破旧的布衣，苦笑一声。 三个月前他还是上市公司高管，如今却成了落魄家族的废物少爷。 更糟糕的是，今天是家族大比，所有旁系子弟都在等着看他的笑话。

错误示例（❌）：
> 在遥远的星河大陆，有着一个名为大周的王朝，王朝下有无数宗门势力...

### 法则2：每500字一个小转折

纯流水账没人看。保持紧张感的方法：
- 每300~500字有一个小冲突/意外/反转
- 每1000字有一个小高潮点
- 每章有一个核心大高潮

### 法则3：对话要推动剧情

对话不是用来填充字数的，对话要：
- 展现人物性格
- 传递关键信息
- 制造/升级冲突
- 暗示伏笔

正确示例：
> "你敢！"周长老眼中杀意毕露。
> 
> 林逸不闪不避，嘴角反而勾起一抹弧度："我不仅敢，我还要当着全族的面，让你跪下来。"

错误示例（❌）：
> "你好，我是林逸。"
> "你好，我是王刚。"
> "今天天气真好啊。"
> "是啊，天气真好。"

### 法则4：感官描写+动作描写

不要只有"他打了一拳"，要写：
```
林逸眼中闪过一丝冷意。
下一瞬，他的拳头已经轰出——
"砰！"
赵无极整个人倒飞出去，重重砸在墙壁上，口吐鲜血。
```

### 法则5：章节末尾必须留钩子

章节结尾要么：
- **悬念钩子**：写到最紧张的地方断掉
- **信息钩子**：揭露一个关键信息
- **情感钩子**：情感高潮点突然停住
- **反转钩子**：意想不到的反转

绝对不能：
- 章节末尾主角悠闲地睡觉
- 章节末尾一个没有悬念的过渡

## 各类场景的写作模板

### 战斗场景

```
【标准战斗模板（3000字）】

铺垫（第500字）：
- 交代为什么打（利益/面子/仇恨）
- 对手嘲讽/挑衅（建立爽感基础）
- 主角内心（冷静/不屑/认真）

第一回合（第600字）：
- 双方试探性出手
- 对手轻视，主角隐藏实力

转折（第700字）：
- 危机出现（对手展示真正实力）
- 主角陷入劣势（增加压迫感）

爆发（第1000字）：
- 主角动用底牌/智慧/成长
- 关键台词（主角说一句话让读者高潮）
- 反杀开始

高潮（第800字）：
- 精彩反杀过程
- 对手震惊/求饶/不甘
- 围观者反应

收尾（第400字）：
- 战斗结束
- 收获/打脸
- 章节钩子

【战斗禁忌】
❌ 不要写一招秒杀（没有压迫感）
❌ 不要写主角全程碾压（没有紧张感）
❌ 不要省略围观者反应（减少爽感）
❌ 不要写完就结束（没有钩子）
```

### 感情场景

```
【感情升温模板】

相遇/接触（第800字）：
- 不期而遇或刻意制造接触
- 第一印象（具体的外貌/行为描写）

互动细节（第1000字）：
- 有趣/暖心/让人心动的细节
- 主角内心的小鹿乱撞
- 配角的神助攻/阻碍

小波折（第700字）：
- 误会/外人打断/突发事件
- 情绪的低谷

情感升温（第500字）：
- 眼神/动作/简短对话
- 暗示性台词
- 章节钩子（下章会更亲密/会有冲突）

【感情戏禁忌】
❌ 不要一上来就表白（没有铺垫）
❌ 不要全是对话（要加动作/心理）
❌ 不要太顺（要有误会/阻碍）
❌ 不要太长（读者会腻，适度即可）
```

### 装逼打脸场景

```
【装逼打脸模板】

铺垫（第600字）：
- 反派/嘲讽者出场，嚣张跋扈
- 主角被轻视/被羞辱
- 围观者同情/嘲笑主角

发酵（第500字）：
- 嘲讽升级
- 主角被逼入困境
- 围观者等着看好戏

转折（第800字）：
- 有人提醒主角身份/实力
- 反派的轻蔑/不信
- 主角淡然开口/出手

高潮（第800字）：
- 实力展示（具体描写）
- 反派震惊/后悔
- 围观者哗然/打脸

收尾（第400字）：
- 反派结局（跪/死/颜面尽失）
- 收获/敬佩目光
- 章节钩子

【装逼打脸禁忌】
❌ 不要让反派降智太多（要有压迫感）
❌ 不要省略围观者反应（爽感减半）
❌ 不要打脸后马上结束（要有后续/余韵）
```

## 章节字数控制

| 章节类型 | 推荐字数 | 说明 |
|---------|---------|------|
| 开局章 | 3000~4000字 | 要交代的信息多，可稍长 |
| 普通章节 | 3000字 | 黄金标准 |
| 战斗高潮章 | 3500~4000字 | 战斗需要更多描写 |
| 感情戏章 | 2500~3500字 | 情感戏不宜太长 |
| 过渡章 | 2000~3000字 | 过渡用，不要太长 |

## 文风保持一致

### 如何保持文风一致

1. **固定主角内心独白方式**：如"林逸心里冷笑"而非时而"他心想"时而"他暗道"
2. **固定语言习惯**：如主角不说"我靠"就说"该死"
3. **固定情绪词**：如用"森冷"而非一会"冰冷"一会"寒冷"
4. **固定叙事节奏**：每章的起伏模式保持稳定

### 避免文风崩坏

每写完一章，检查：
- [ ] 主角说话风格一致
- [ ] 战斗描写风格一致
- [ ] 情感描写风格一致
- [ ] 叙事节奏一致
- [ ] 没有用词突变
