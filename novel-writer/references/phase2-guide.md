# Phase 2 参考：分卷大纲 & 角色设定指南

## 角色设定深度指南

### 主角人物卡模板

主角是全书的灵魂，人物卡必须深度完整：

```markdown
## {主角姓名}

### 基础信息
| 字段 | 内容 |
|------|------|
| 姓名 | {真名/化名} |
| 性别 | 男/女 |
| 年龄 | {年龄段} |
| 身份 | {开篇身份} |
| 外貌 | {关键外貌特征，3~5句话} |

### 性格特质
- **核心性格**：（如：外冷内热/亦正亦邪/阳光热血）
- **性格优点**：（3~5个，如：重情重义、心思缜密、坚韧不拔）
- **性格缺点**：（2~3个，如：冲动易怒、不善表达）

### 背景故事
**（按时间线描述主角的过往经历）**
- 童年/起源：（0~50字）
- 关键转折：（100~200字）
- 开篇状态：（50~100字）

### 核心矛盾/内心冲突
**（主角内心最深层的矛盾，是角色弧线的驱动力）**
- 矛盾点1：...
- 矛盾点2：...

### 成长弧线
- **起点**：（开篇时的状态）
- **转折点**：（哪个事件让主角改变）
- **终点**：（结尾时的状态）

### 语言风格
- **说话方式**：（如：简洁直接/文绉绉/带方言/痞气）
- **标志性台词**：（主角可能会说的经典台词风格）

### 金手指/核心优势
- **类型**：（系统/老爷爷/特殊体质/知识/其他）
- **具体表现**：（描述金手指的具体功能和限制）

### 与其他角色的关键关系
| 角色 | 关系类型 | 关系描述 |
|------|----------|---------|
| 配角名 | 亲人/爱人/友人/敌人 | 简述 |
```

```markdown

## 卷结构

### 卷首（第1~3章）

## 伏笔记录
| 伏笔 | 埋入章节 | 对应回收章节 |
|------|----------|------------|

## 卷末总结/反思
（写完后复盘：这卷写得好不好？有哪些问题？）
```

### 大纲撰写注意事项

1. **每章必须有"核心事件"**：不能写"这章是过渡章"，要有具体剧情
2. **章节之间要有因果链**：A章→B章→C章要有逻辑联系，不能跳跃
3. **每3~5章设计一个小高潮**：不能让读者觉得无聊
4. **提前规划"名场面"**：至少每5~10章有一个让人印象深刻的名场面
5. **反派要有压迫感**：让读者担心主角能不能赢

## 网文卷结构规律

### 常见的卷节奏类型

**升级流卷节奏**（适合玄幻/修仙）：
```
开篇压制 → 逆境成长 → 首次胜利 → 更大挑战 → 卷高潮 → 境界突破
```

**都市异能卷节奏**（适合都市/系统流）：
```
日常/装逼铺垫 → 强敌出现 → 被压制 → 智取/爆发 → 胜利 → 收获/悬念
```

**感情线卷节奏**（适合言情/恋爱文）：
```
相遇 → 误解/冲突 → 了解 → 暧昧升温 → 表白/感情确定 → 外部阻碍
```

### 每卷字数分配建议

| 部分 | 占卷比例 | 字数（30万字卷） |
|------|---------|----------------|
| 开篇（前3章） | ~3% | ~1万字 |
| 铺垫（4~8章） | ~15% | ~4.5万字 |
| 主线展开（9~N-3章） | ~70% | ~21万字 |
| 卷高潮（倒数3章） | ~7% | ~2万字 |
| 悬念过渡（最后1章） | ~1% | ~0.5万字 |
