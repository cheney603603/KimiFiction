# Phase 3 参考：章节细纲撰写指南

## 细纲的核心理念

细纲是"骨骼"，正文是"血肉"。好的细纲让写作变得流畅自然，差的细纲让写作变成流水账填充。

**好的细纲标准**：
- 有具体的场景、动作、对话要点（不是模糊的"主角打boss"）
- 有情感变化的节点（情绪的起伏）
- 有语言/动作的细节描写提示
- 考虑了章节之间的节奏感

## 章节细纲模板

```markdown
# 第{XXX}章 {章节名}

## 基本信息
| 字段 | 内容 |
|------|------|
| 章节字数 | {3000~4000字} |
| 章节类型 | 开局/日常/战斗/感情/转折/升级 |
| 重要程度 | ★★★☆☆（1~5星） |
| POV视角 | 主角/配角（如涉及） |

## 章节核心任务
**（这一章必须完成的一件事，不能省略）**
> 核心任务：XXX

## 情节点（场景展开）

### 场景①：{场景名}
- **地点**：{具体地点}
- **人物**：{出场人物}
- **发生什么**：
  - {具体动作/事件描述}
  - {关键对话1}："xxx。"
  - {关键对话2}："xxx。"
- **情绪基调**：（如：紧张、轻松、悲伤）

### 场景②：{场景名}
- **地点**：
- **人物**：
- **发生什么**：
  - ...
- **情绪基调**：

### 场景③：{场景名}
...

## 章节亮点设计
**（这一章最能让读者记住的点）**

### 记忆点
- **点1**：（如：主角一句经典台词）
- **点2**：（如：某个让人震撼的场景描写）

### 氛围营造
（如：战斗时的紧迫感/恋爱时的甜蜜氛围，用什么手法营造）

## 伏笔/钩子记录
| 类别 | 内容 | 后续章节 |
|------|------|---------|
| 伏笔 | {描述} | {后续回收章节} |
| 钩子 | {悬念描述} | {下章/未来章节} |

## 章节与前后章节的衔接
- **承接上文**：第{XXX}章，{什么事件}后，{主角处于什么状态}
- **引发下文**：本章结束后，主角即将{做什么事}，引出{什么冲突}

## 金句/关键台词记录
```
"xxx。" —— {角色名}（场景①）
"xxx。" —— {角色名}（场景②）
```

## 写作注意事项
- 【注意】：{需要特别注意的点}
- 【避免】：{需要避免的问题，如：不要写成流水账}
- 【参考】：{参考某作品的某段落写法}
```

## 各类章节的细纲侧重点

### 开局章（第1章）细纲要求

第1章是最重要的，必须精细设计：

```
必须包含：
✅ 主角出场（让读者认识主角）
✅ 核心冲突/矛盾出现（让读者知道主角面临什么困境）
✅ 核心设定/金手指亮相（让读者知道主角有什么优势）
✅ 章节末尾的钩子（让读者想看第2章）

禁止出现：
❌ 过多的世界观介绍（会让读者无聊）
❌ 过多配角出场（分散注意力）
❌ 平铺直叙没有冲突
❌ 章节末尾平淡无奇
```

### 战斗章细纲要求

```
✅ 战斗前铺垫（为什么打？双方状态）
✅ 战斗过程（起承转合，不是纯打，有心理描写）
✅ 转折点（通常是主角的智慧或隐藏底牌）
✅ 战斗结果
✅ 战后余波（收获/后续影响）

战斗节奏示例（3000字章节）：
- 铺垫：500字（对手嘲讽，主角被压制）
- 第一次交锋：500字（试探性攻击）
- 被逼入绝境：500字（展示危机感）
- 转折：500字（底牌/智慧/成长）
- 高潮反击：1000字（精彩反杀）
- 战后：500字（收获/嘲讽者打脸）
```

### 感情线章细纲要求

```
✅ 情感铺垫（什么事件触发情感）
✅ 暧昧/互动细节（甜蜜感的来源）
✅ 情感冲突/误会（增加曲折）
✅ 情感升温/确认（明确关系进展）
✅ 章节钩子（如何留住读者）

感情戏避免：
❌ 太快确认关系（没意思）
❌ 全是对话（要有动作/环境/心理）
❌ 过度甜腻（适时加入冲突调味）
```

### 升级/收获章细纲要求

```
✅ 升级的原因（为什么能升级）
✅ 升级过程的描写（感受上的变化）
✅ 新能力/收获的具体展示（读者要知道有什么好处）
✅ 能力带来的心态变化
✅ 引出下一步挑战（升级不是终点）
```

## 章节间的节奏设计

### 连续章节节奏搭配建议

| 前章类型 | 后章类型 | 效果 |
|---------|---------|------|
| 战斗高潮章 | 收获总结章 | 爽感延续，让读者消化 |
| 战斗高潮章 | 日常过渡章 | 张弛有度，不让读者疲劳 |
| 日常铺垫章 | 冲突爆发章 | 冲突感更强 |
| 情感升温章 | 外部阻碍章 | 增加曲折感 |
| 升级收获章 | 新挑战出现章 | 升级有意义 |

### 每章内部节奏（3000字示例）

```
段落1（300字）：开场，交代场景，引出事件
段落2（300字）：冲突/事件升级
段落3（400字）：关键对话/动作
段落4（300字）：小转折/小高潮
段落5（300字）：情绪/氛围铺垫
段落6（400字）：高潮冲突
段落7（300字）：结果/情绪释放
段落8（300字）：章节钩子/悬念
段落9（400字）：伏笔/为下章铺垫
```

## 细纲审核清单

在提交细纲给用户前，AI自我检查：

- [ ] 每章有明确的核心任务（不是"过渡"）
- [ ] 章节末尾有钩子（读者会想继续看）
- [ ] 有具体的对话/动作（不是模糊的"发生冲突"）
- [ ] 情绪基调明确（读者能感受到情绪）
- [ ] 与前后章有衔接（不是孤立存在的章节）
- [ ] 主角有内心变化/成长（不是工具人）
- [ ] 重要角色有记忆点设计
